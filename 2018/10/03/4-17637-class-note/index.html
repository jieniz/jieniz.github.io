17-637 Web Application Development 笔记

Posted on 2018-10-03 | In 17-637 Web Application Development |

DOM

document object model: a tree of nodes that represents the HTML document

Navigate DOM with JS

Given an element node,
navigate the tree:

  • node.parentNode, node.firstChild, node.lastChild, node.childNodes
  • node.nextSibling, node.previousSibling

access/modify properties:

  • nodeName, nodeType, attributes, getAttribute(attr), id, innerHTML
  • offsetLeft, offsetTop, offsetParent

get a DOM node:

  • document.getElementById(id) / ByTagName(name) / ByClassName(name)
  • 遍历DOM树, id 唯一，tag和class一般不唯一，返回 array of nodes

Javascript

server-side web application:

  • web application 只在web server中运行
  • browser 每次只能发送一个request，每收到一个response都要重新展示页面re-render/re-load

client-side web application:

  • web application 可以在broswer和server中运行
  • 用户可以在browser中与web app交互，不需要通过网络连接server，对response响应快，不需要重新加载页面

client-side web programming language: Javascript

  • get JS to the browser:
    • in HTML page: <script type="text/javascript"></script>
    • in external file: <script src="foo.js" type="text/javascript"></script>
  • respond to in-browser events: onclick, onload, onfocus, onblur…

Tutorial: https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide
