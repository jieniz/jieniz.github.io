4.1 Route Between Nodes

Input

(directed) Graph g, Node start, Node end

Output

判断两点之间是否有通路

Pseudo Code

  • enum State { unvisited, visiting, visited }
  • 遍历graph, 设置node.state=State.unvisited, 其中start设为visiting
  • 把visiting的node加入queue
  • while(!q.isEmpty())
    • 取出第一个node n, 如果n不为空
      • 遍历Node v : n.getAdjacent()
        • 如果v == end, return true
        • 如果v.state = unvisited, 把state变为visiting并加入queue
      • 设置n的状态为visited
  • return false

4.2 Minimal Tree

Input

有序数组，元素唯一

Output

binary search tree with minimal height

Pseudo Code

迭代，树的前序遍历

  • createBST(int arr[], int start, int end)
    • 如果start > end, 停止return null
    • 中点即为root，mid = (start + end ) / 2，new TreeNode(arr[mid])
    • n.left = createBST(arr, start, mid - 1)
    • n.right = createBST(arr, mid + 1, end)
    • return n

4.3 List of Depths

Input

Binary tree, Depth = d

Output

d个linkedlist, 每个list包含在depth d的所有nodes

Pseudo Code

迭代，树的遍历

  • 新建ArrayList<LinkedList<TreeNode>> lists
  • createList(TreeNode root, ArrayList<LinkedList<TreeNode>> lists, int level)
    • base case: if root == null, return;
    • 新建LinkedList list = null, 根据level得到着一层的list
      • 如果lists.size()==level, 说明这一次的list还没有初始化，lists.add(list)
      • else: 从lists中取出list=lists.get(level)
    • 把此时的node加入list中
    • createList(root.left, lists, level+1)
    • createList(root.right, lists, level+1)
  • return lists

4.4 Check Balanced

Input

binary tree

Output

check if it is balanced: 任一节点的两个subtrees的高度差不超过1

Pseudo Code

迭代+树的遍历

  • int checkHeight(TreeNode node)
    • base case: node == null, return -1
    • 左边迭代leftHeight = checkHeight(node.left), if leftHeight == MIN, return MIN
    • 右边迭代rightHeight = checkHeight(node.right), if rightHeight == MIN, return MIN
    • diff是两者的差，>1 return MIN, else: return Math.max(leftHeight, rightHeight) + 1
  • return checkHeight(root) != MIN

4.5 Validate BST

Input

Binary tree

Output

Check if it is a binary search tree: left <= cur < right 整体，不仅是单个节点

Pseudo Code

迭代检查每个节点的区间

  • int checkBST(TreeNode n, int min, int max)
    • base case: 树到底了，n == null, return true
    • 检查自己太小或太大：min != null && n.data < min 或者 max != null && n.data > max, return false
    • 检查子节点：!checkBST(n.left, min, n.data) 或者 !checkBST(n.right, n.data, max), return false
    • return true
  • return check(node, null, null) // null表示最大或最小

4.6 Successor

Input

TreeNode n

Output

n的下一个节点(in-order, 左中右顺序遍历）

Pseudo Code

如果n有右子节点，返回右子节点树中最左边的节点；否则，一直向上找到子节点在父节点的左边

  • n.right != null
    • while(n.left!=null) n = n.left(), 返回n
  • n.right == null
    • 父节点 p = n.parent, while(p!=null && p.left!=n)一直往上，n=p, p=p.parent
    • 如果循环停止，有两种可能
      • p==null, 说明到头了n没有下一个节点
      • p.left==n, 说明找到n在p的左子节点的情况，p就是要找的node
    • return 父节点p

4.7 Build Order

Input

一组String[] projects, 一组String[][] dependencies（第一个project必须在第二个project前build）

Output

返回build order，如果没有返回error

Pseudo Code

  • Project类：
    • field: ArrayList<Project> children, HashMap<String, Project> map, String name , int dependencies, enum State, State state = Blank,
    • constructor: Project(String n) { name = n;}
    • methods:
      • getName(), getChildren(), getDepen(), increDepen(), decreDepen(), getState(), setState()
      • addNeighbor(Project): 如果!map.containsKey(node.getName()), 把node加入children, map，并且increDepen
  • Graph类：
    • fields: ArrayList<Project> nodes, HashMap<String, Project> map
    • methods:
      • getNode(): nodes
      • createNode(name): 如果!map.containsKey(name), 新建new Project(name), 放入nodes和map，return 新建的project
      • addEdge(start, end): 新建Project start和end，start.addNeighbor(end)
  • 以project为点，dependency为线，建表
    • for(proj:projects) graph.createNode(proj)
    • for(dep:dependencies) graph.addEdge(dep[0],dep[1])
  • 新建stack用来存放结果，遍历所有project, 如果state是BLANK
    • 如果!doDFS(Project proj, Stack stack), return null
      • state == PARTIAL, 陷入循环return false
      • state == BLANK, state变为PARTIAL，
        • 遍历children，如果!doDFS(child, stack), return false;
        • state变为COMPLETE，入栈
      • return true

4.8 First Common Ancestor

Input

一个二叉树中的根root, 两个节点 p, q

Output

第一个共同的父节点（要求：不存储多余节点）

Pseudo Code

  • Ancestor类：
    • fields: Node, isAncestor
    • constructor: Ancestor(Node n, boolean isAncestor)
  • 如果ancHelper(root, p, q).isAncestor, return node, 否则return null
  • Method: Ancestor ancHelper(TreeNode root, p, q)
    • if root == null, return new Ancestor(null, false)
    • 三个点是同一个：if root == p && root == q, return new Ancestor(root, true)
    • 向左：rx = ancHelper(root.left, p, q), 如果rx是ancestor, return 这个点
    • 向右：ry = ancHelper(root.right, p, q), 如果ry是ancestor, return 这个点
    • 如果都不为空说明找到了 rx.node != null && ry.node != null, return new Ancestor(root, true)
    • 如果root是q或者p，表示我们现在在p或q，且另一个在subtree，说明也找到，return new Ancestor(root, rx.node != null || ry.node != null)
    • else: return new Ancestor(rx.node != null ? rx.node : ry.node, false)

4.9 BST Sequences

Input

Binary search tree（元素唯一）的根节点 TreeNode root

Output

所有可能生成这个BST的array, ArrayList<LinkedList<Integer>>

Pseudo Code

  • allSeq(TreeNode root)
    • 新建ArrayList result存放结果，如果node是空，向数组中加入一个空的linkedlist，然后返回arraylist
    • 新建LinkedList<Integer> prefix，把node加入list
    • 左右树分别迭代，遍历树：
      • ArrayList<…> leftSeq = allSeq(root.left)
      • ArrayList<…> rightSeq = allSeq(root.right)
    • 循环所有生成的list，调用weaveList方法，把所有list按照由左向右的方式交织
      • for(left : leftSeq)
        • for(right : rightSeq)
          • 建ArrayList<Link…> weaved
          • 传入方法weaveLists(left, right,weaved, prefix)
          • 把每一个weaved 加入result
  • weaveLists(LinkedList first, LinkedList second, ArrayList results, LinkedList prefix)
    • 如果fisrt或second其中一个size为0
      • result=prefix.clone()，然后add first, second,
      • 把result加入results中，返回
    • 依次拿出first中的每个元素，迭代weaveLists
      • head=first.removeFirst(), 把head加入prefix
      • 迭代weaveLists(first, second, results, prefix)
      • 把head放回去，prefix.removeLast(), first.addFirst(head)
    • 依次拿出Second中的每个元素，迭代weaveLists
